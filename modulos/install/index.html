Creación de BD y User para Iwana

Para una correcta configuración de Iwana, necesitamos que nos proporciones usuario y contraseña con privilegios elevados para poder crear la Bd y User que utilizara Iwana.

Para una correcta configuración inicial y parametrización de IWANA, siga estos pasos.


  • 1. Instalé y configuré un servidor web basado en php en su última versión preferiblemente con apache y Mysql o Maria DB o su motor de base de datos preferido.
  • 2. Copie el paquete Iwana a la raíz de su servidor web.
  • 3. Instale y configure un servidor FTP.
  • 4. Tenga a la mano la siguiente información:
    • Datos de su entidad como: Nit, razón social, dirección, teléfonos, email y dirección web.
    •  Como desearía generar los números de radicado, opciones disponibles YYYYMMDD-#####, COD DEPENDENCIA-YYYYMMDD-##### o COD DEPENDENCIA-#####.
    •  como desea imprimir el rotulo del radicado, opciones disponibles: Impresora de Tecket o Documento físico.
    •  Organigrama “Configuración de dependencias, oficinas”
    •  Listado maestro de cargos.
    •  Funcionarios de la institución “Configure al menos un funcionario”.
